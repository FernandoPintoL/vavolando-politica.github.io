Política de Privacidad

Última actualización: Abril 2026

1. Introducción

Va Volando ("la Empresa", "nosotros" o "nuestro") opera la aplicación móvil Va Volando ("la Aplicación"). Esta página informa sobre nuestras políticas de recopilación, uso y divulgación de datos personales cuando utiliza nuestra Aplicación, así como las opciones que tiene asociadas con esos datos.

2. Datos que Recopilamos

La Aplicación recopila los siguientes tipos de datos:

2.1 Datos de Ubicación

  • Ubicación Precisa (GPS): Recopilamos su ubicación exacta con su consentimiento para proporcionar servicios de mapeo, seguimiento de entregas y optimización de rutas.
  • Ubicación Aproximada: Información de red para determinar su ubicación aproximada.

2.2 Datos de Cámara

  • Acceso a la cámara para capturar fotos de productos, documentos y códigos QR.
  • No se transmiten automáticamente; usted controla qué fotos capturar y compartir.

2.3 Datos de Contacto

  • Números de teléfono para realizar y recibir llamadas en el contexto de sus transacciones comerciales.
  • Información de contacto de clientes con consentimiento explícito.

2.4 Datos de Notificaciones

  • Token de notificaciones push para enviarle actualizaciones sobre pedidos y entregas.

2.5 Datos Biométricos

  • Autenticación de huella dactilar para mejorar la seguridad de la Aplicación.

2.6 Datos de Almacenamiento

  • Acceso a imágenes almacenadas para seleccionar fotos en la Aplicación.

2.7 Otros Datos

  • Información de uso de la Aplicación, errores y datos de rendimiento.
  • Dirección IP, tipo de dispositivo, sistema operativo.

3. Cómo Usamos Sus Datos

Utilizamos los datos recopilados para:

  • Proporcionar y mejorar los servicios de la Aplicación
  • Procesar transacciones y entregas
  • Optimizar rutas de distribución
  • Comunicaciones relacionadas con el servicio (notificaciones, actualizaciones)
  • Autenticación y seguridad
  • Análisis y mejora del rendimiento de la Aplicación
  • Cumplir con obligaciones legales

4. Base Legal para el Procesamiento

Procesamos sus datos personales basándonos en:

  • Su Consentimiento: Para ubicación, cámara y datos biométricos, recopilamos su consentimiento explícito.
  • Necesidad Contractual: Para procesar su solicitud y proporcionar servicios.
  • Obligación Legal: Para cumplir con requisitos legales y regulatorios.

5. Seguridad de Datos

Implementamos medidas técnicas y organizacionales para proteger sus datos personales contra acceso no autorizado, alteración, divulgación o destrucción. Sin embargo, ningún método de transmisión por Internet es 100% seguro. No podemos garantizar la seguridad absoluta de sus datos.

6. Retención de Datos

Retenemos sus datos personales solo mientras sea necesario para proporcionar los servicios o como requiera la ley. Datos de ubicación y fotos se almacenan según sea necesario para las transacciones correspondientes. Puede solicitar la eliminación de sus datos en cualquier momento.

7. Sus Derechos

Usted tiene derecho a:

  • Acceder a los datos personales que mantenemos sobre usted
  • Corregir datos inexactos
  • Solicitar la eliminación de sus datos
  • Revocar su consentimiento en cualquier momento
  • Rechazar permisos en la Aplicación o en la configuración de su dispositivo

8. Permisos de la Aplicación

La Aplicación solicita los siguientes permisos que son opcionales. Puede rechazarlos en la configuración de su dispositivo. El rechazo de ciertos permisos puede limitar la funcionalidad de la Aplicación:

  • Ubicación (GPS): Para mapeo y seguimiento de entregas
  • Cámara: Para capturar fotos y códigos QR
  • Galería/Almacenamiento: Para seleccionar imágenes
  • Notificaciones: Para alertas de pedidos y entregas
  • Teléfono: Para realizar llamadas a clientes
  • Biometría: Para autenticación de seguridad

9. Cambios a Esta Política

Podemos actualizar esta Política de Privacidad ocasionalmente. Le notificaremos sobre cambios materiales mediante un aviso en la Aplicación o por correo electrónico. Su uso continuado de la Aplicación después de tales notificaciones constituye aceptación de los cambios.
